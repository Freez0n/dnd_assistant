# D&D Ассистент
**D&D Ассистент** — это десктопное приложение для автоматизации управления игровым процессом в Dungeons & Dragons (5e). Проект создан как дипломный проект для упрощения ведения кампаний, управления персонажами, отслеживания боевых встреч и генерации случайного контента. Приложение обеспечивает удобный интерфейс для мастера и игроков, централизуя все необходимые инструменты в одной системе.

## Функционал и возможности

* **Управление персонажами**: Создание, редактирование листов персонажей и автоматический расчет характеристик.
* **Боевой трекер**: Инструмент для отслеживания инициативы, здоровья участников и состояний в бою.
* **Бестиарий**: Личный справочник монстров с системой фильтрации по уровню опасности (CR).
* **Генераторы**: Автоматическая генерация НПС, завязок квестов, сокровищ и случайных предметов.
* **Система бросков кубиков**: Встроенный движок для вычисления формул бросков с поддержкой преимуществ/помех и сохранения пользовательских формул.
* **База знаний**: Система категоризированных заметок для фиксации сюжета, локаций и секретов кампании.
* **Аутентификация**: Управление пользовательскими сессиями и безопасное хранение данных.

## Стек проекта

* **Язык**: Python 
* **Интерфейс (GUI)**: CustomTkinter
* **База данных**: MySQL (используется `mysql-connector-python`)
* **Архитектура**: Клиент-серверное взаимодействие с БД.

## Как запустить проект

1. **Подготовка**:
* Убедитесь, что у вас установлен **Python 3.x**.
* Установите необходимые зависимости:
```bash
pip install customtkinter mysql-connector-python bcrypt

```




2. **Настройка и запуск**:
* Откройте файл `main.py` и в строке инициализации `DatabaseManager` укажите пароль от вашего локального сервера MySQL:
```python
self.db = DatabaseManager(password='ВАШ_ПАРОЛЬ')

```




* После запуска откроется окно приложения. Если это ваш первый запуск, зарегистрируйтесь — система автоматически заполнит базу данных начальными данными для вашего пользователя.


3. **Запуск**:
* Перейдите в корневую папку проекта в терминале.
* Запустите главный файл приложения:
```bash
python main.py

```


* При первом входе пройдите регистрацию — система автоматически заполнит начальные данные для вашего пользователя.
